# Aura 使用说明

## 支持范围

当前版本：
- 浏览器：Chrome
- 视频站点：腾讯视频

## 安装

1. 进入 Aura 项目目录
2. 执行：

```bash
npm run build
```

3. 在 Chrome 打开 `chrome://extensions`
4. 开启开发者模式
5. 选择“加载已解压的扩展程序”
6. 选择 `dist/`

## 基础使用

1. 打开腾讯视频播放页
2. 开始播放影视剧内容
3. 点击 Aura 插件图标打开 popup
4. 查看：
   - 当前剧集
   - 推荐主题
   - 当前模式
5. 调节：
   - 开关
   - 强度
   - 手动主题

## 快捷键

- 主快捷键：`Ctrl + Shift + A`
- 辅助快捷键：`Option + Command + A`

## 主题切换

### 自动推荐
Aura 会优先根据剧名推荐主题：
- 剧集专属主题优先
- 类型主题次之
- 默认主题兜底

### 手动主题
也可以在 popup 中手动切换：
- 腾讯默认 Aura
- 古装仙侠
- 都市爱情
- 悬疑刑侦
- 逐玉 · 专属 Aura

## 调试信息

点击 popup 中的“调试信息”按钮，可以展开：
- 实际主题
- video 数量
- 容器识别状态
- 黑边信息
- 是否走容器兜底

## 常见问题

### 1. 插件没有显示效果
- 确认已在腾讯视频页面
- 刷新页面
- 确认插件开关已开启
- 把强度调高到 50 以上再观察

### 2. 快捷键无效
- 先确认页面或 popup 已获得焦点
- 优先使用 `Ctrl + Shift + A`
- 如果仍无效，尝试到 Chrome 扩展快捷键页重新绑定

### 3. 主题没有自动命中
- 刷新当前播放页
- 确认剧名在页面中可见
- 打开调试信息查看当前识别剧名
