# Aura 路线图

## V1 目标

做成一个用户可安装、可在腾讯视频稳定使用的观影氛围层插件。

### V1 必达
- 腾讯视频窗口态黑边覆盖稳定
- 腾讯视频全屏态黑边覆盖稳定
- 快捷键稳定切换
- popup 产品化
- 主题系统基础可用
- 至少 1 个剧集专属主题

## V1.1

- Theme Registry 刷新链彻底稳定
- 设置页
- 最近使用主题
- 更多剧集专属主题

## V1.5

- 远程主题清单
- 在线主题分发
- 自动拉取热门剧 Aura 包

## V2

- 多站点支持（爱奇艺 / 优酷 / B站）
- 云端主题生产链
- 用户个性化 Aura
- 收藏与分享
