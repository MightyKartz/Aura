# Aura 路线图

## V1 目标

做成一个用户可安装、可在腾讯视频稳定使用的角落主题挂件插件。

### V1 必达
- 腾讯视频窗口态角落挂件稳定
- 腾讯视频网页全屏 / 原生全屏稳定
- 快捷键稳定切换，关闭后无残留
- popup 设置与页面状态一致
- 主题系统基础可用
- 默认 / 悬疑 / 古偶 / 热血 四套皮肤可用

## V1.1

- 更细的亮场 / 广告态观感优化
- 更好的素材模板与预览体系
- popup 中更清晰的状态表达
- 为第二站点 adapter 做代码预留

## V1.5

- 远程皮肤清单
- 在线主题分发
- 热门剧皮肤活动

## V2

- 多站点支持（爱奇艺 / 优酷 / B站）
- 云端主题生产链
- 用户个性化 Aura
- 收藏与分享
