# Aura 皮肤素材尺寸规范 V2

## 目标

这份规范服务于当前 Aura 的角落挂件主线。每套皮肤只需要两张必需图：

1. 左上角题材角标
2. 右下角主角色挂件

可选再补一张预览图，方便后续在文档或 popup 中展示。

---

## 推荐出图尺寸

## 1. 左上角题材角标

### 推荐比例

- `1:1`

### 推荐生成尺寸

- 首选：`1536 x 1536`
- 备选：`1024 x 1024`

### 用途

- 对应文件：`skin-<genre>-top-left.png`
- 用于播放器左上角的小型题材角标

### 构图要求

- 主体集中在画布左上 `60% x 60%` 区域
- 右下区域尽量留空
- 不要做成完整海报
- 不要有完整矩形底板
- 适合做轻装饰、徽章、发饰、花瓣、线索板、火花等

---

## 2. 右下角主角色挂件

### 推荐比例

- 首选：`4:5`
- 备选：`3:4`

### 推荐生成尺寸

- 首选：`1536 x 1920`
- 备选：`1536 x 2048`
- 若模型较弱，可先用：`1024 x 1280`

### 用途

- 对应文件：`skin-<genre>-bottom-right.png`
- 用于播放器右下角的小猫主角色挂件

### 构图要求

- 主体落在画布右下 `70% x 75%` 区域
- 左上留白，避免压到视频中心
- 更适合“半身 / 探头 / 小挂件”而不是完整站姿
- 轮廓要干净，避免过多细小碎饰
- 不要做成满画幅的大型背景插画

---

## 3. 可选预览图

### 推荐比例

- `16:9`

### 推荐生成尺寸

- `1536 x 864`
- 或 `1920 x 1080`

### 用途

- 对应文件：`skin-<genre>-preview.jpg`
- 用于文档、后续 popup 或宣传图，不进入当前运行时主链

---

## 当前项目最适合的出图组合

每套皮肤建议先出这 2 张：

1. 左上角标：`1536 x 1536`
2. 右下角色：`1536 x 1920`

如果时间够，再补：

3. 预览图：`1536 x 864`

---

## 输出格式建议

## 运行时素材

- 最终格式：`PNG`
- 背景：透明
- 命名：
  - `skin-suspense-top-left.png`
  - `skin-suspense-bottom-right.png`
  - `skin-ancient-top-left.png`
  - `skin-ancient-bottom-right.png`
  - `skin-hotblood-top-left.png`
  - `skin-hotblood-bottom-right.png`

## 预览图

- 最终格式：`JPG` 或 `PNG`
- 背景可保留

---

## AI 出图注意事项

1. 先生成“大母图”，不要直接生成很小的最终图
2. 角色和装饰尽量单主体，避免多角色拥挤
3. 最终进项目时最好做一次抠图，保留透明背景
4. 同一类皮肤应保持统一画风、描边、光感和色相
5. 不要让角标和角色都太亮，否则亮场里会抢戏

---

## 三类皮肤视觉建议

## 悬疑

- 关键词：冷蓝、黄铜、雾气、放大镜、线索板、侦探猫
- 左上：放大镜 / 线索卡 / 冷雾徽章
- 右下：警觉、侧头、认真观察的小猫

## 古偶

- 关键词：暖粉、浅金、玉饰、流苏、月华、花瓣
- 左上：团扇 / 花枝 / 金月角标
- 右下：温柔、圆润、带古风头饰的小猫

## 热血

- 关键词：橙红、金光、速度线、徽章、火花、冲劲
- 左上：速度角标 / 能量徽章 / 火花符号
- 右下：更有动势、更有冲劲的小猫

---

## 一句话执行建议

现在最适合你立刻去生成的，就是每个皮肤先出：

- `1 张 1:1 左上角标`
- `1 张 4:5 右下角色`

这套组合最符合现在 Aura 的运行时和 AI 生图模型的工作方式。
