# Aura 皮肤执行清单 V1

这份清单只服务于当前 Aura 的双角挂件主线，目标是把每套皮肤拆成清晰的两个角色：

1. 左上角：题材或品牌角标
2. 右下角：主角色挂件

## 默认皮肤还要补哪些画面问题

### 1. 角色分工不够明确

- 当前默认皮肤的左上和右下都在画“同一只可爱猫”，差异主要靠道具变化，导致两个角的职能不够清楚。
- 默认皮肤应该更像：
  - 左上：Aura 官方品牌角标
  - 右下：Aura 官方陪看角色

### 2. 左上素材还不够“角标化”

- 左上不应该是一张缩小版角色图。
- 更适合做成：
  - 半徽章
  - 星月标记
  - Aura 品牌符号
  - 只露出一点猫耳或猫爪的轻装饰

### 3. 默认角色的情绪表达还可以更稳定

- 默认皮肤的右下角色应该传达“陪伴、温柔、安心”，而不是过于兴奋或过于夸张。
- 最适合默认皮肤的表情：
  - 微笑
  - 轻微惊喜
  - 放松陪看

### 4. 左上和右下的视觉语言需要统一

- 两张图要共享同一套：
  - 描边粗细
  - 光感
  - 暖白主色
  - 星点装饰密度

## 新悬疑皮肤应该怎么画

### 核心气质

- 不是恐怖，而是“警觉、观察、冷静、会发现细节”
- 关键词：
  - 冷蓝
  - 黄铜
  - 线索板
  - 雾感
  - 夜色
  - 侦探猫

### 情绪方向

- 右下角色应该像正在陪观众一起“盯线索”
- 表情建议：
  - 认真盯看
  - 微微侧头
  - 有一点警觉
  - 不要做成凶狠攻击型

### 颜色建议

- 主色：雾蓝 / 冷灰蓝
- 点缀：黄铜金 / 证据板暖黄
- 亮点：少量冷白高光

## 左上和右下各自该出什么构图

### 默认皮肤

#### 左上

- 更适合：
  - 星月徽章
  - Aura 字形感角标
  - 半露出的猫耳 / 猫爪 / 星星
- 不适合：
  - 大头完整猫脸
  - 画面中心堆满主体
  - 满铺矩形底板

#### 右下

- 更适合：
  - 探头式半身小猫
  - 趴边陪看
  - 有一点点魔法感的小道具
- 不适合：
  - 完整站立全身
  - 太多小碎装饰
  - 颜色过亮导致亮场里发白

### 悬疑皮肤

#### 左上

- 更适合：
  - 放大镜 + 证据板图钉
  - 冷雾徽章
  - 猫耳侦探帽角标
- 不适合：
  - 做成普通 logo
  - 细节太碎，缩小后看不清

#### 右下

- 更适合：
  - 侦探猫从右下角探出
  - 一只爪子举着放大镜
  - 身后少量线索卡、警戒光点或雾感光晕
- 不适合：
  - 写实侦探人物
  - 背景太厚
  - 大面积深黑导致压画面

## 最容易踩坑的地方

### 1. 把左上画成第二个主角

- 左上必须“小而准”，不是再放一个角色大头。

### 2. 右下信息量太大

- 右下只保留一个主动作和一个主道具，不要堆帽子、披风、徽章、火花、字牌一起上。

### 3. 亮场可见性不足

- 不要只靠低饱和冷色。
- 无论默认还是悬疑，都要保留清晰描边和局部高光。

### 4. 透明边界没处理好

- 轮廓外不要残留脏灰边、黑边和硬阴影。
- 进入视频时，轮廓必须干净。

### 5. 左右风格不统一

- 同一套皮肤的左上和右下不能像来自两个不同模型。
- 统一画风比单张图好看更重要。

- 右下继续保留当前暖白小猫主角色方向，只做角色定位收口，不改成别的题材

- 优先做 `悬疑猫`

- 右下做“侦探猫探头 + 放大镜观察”主角色挂件
